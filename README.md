# Career Twin

AI-powered Digital Career Mentor

## Problem

Students often lack personalized career guidance and struggle to identify the right skills, roadmap, and opportunities.

## Solution

Career Twin acts as an AI mentor that:
- Generates personalized career plans
- Suggests skills to learn
- Recommends projects
- Guides students toward industry roles

## Features

- AI Career Guidance
- Personalized Roadmaps
- Skill Gap Analysis
- Career Q&A Assistant

## Technology Stack

- Manus AI
- Google Gemini API
- React
- JavaScript
- HTML/CSS

## Future Scope

- Resume Analysis
- college and career role suggestion based on profile and intrest
- Interview Preparation
- Internship Recommendations
- LinkedIn Profile Optimization
